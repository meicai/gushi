# 🏯 墨韵诗林 — 少儿古诗背诵助手

一款专为少儿设计的古诗词启蒙与陪伴式背诵 Web 应用。采用极简水墨国风 + 宣纸护眼配色，结合智能记忆闭环系统，让古诗学习变成一件循序渐进的趣事。

## ✨ 核心亮点

| 体验维度       | 特性说明                                                                                                    |
| -------------- | ----------------------------------------------------------------------------------------------------------- |
| **诗词图谱**   | 学前 ~ 六年级共 95 首精选古诗，涵盖小学通用课标；支持按“节日/节气”专题或“田园/思乡”等标签维度多维探索。     |
| **学习闭环**   | 根据算法追踪学习记录：提供“今日推荐”、“智能复习”及“继续学习”入口，不再盲目背诵。                            |
| **四级背诵**   | 学习详情页右上方提供 4 档递进背诵模式：【原诗】 -> 【首字提示】 -> 【核心关键字提示】 -> 【全遮挡测试】。   |
| **交互辅助**   | 每个汉字均支持开关顶部拼音（精确到字级多音字）；支持通俗易懂的白话译文对照；点击即可逐行或者全文 TTS 朗读。 |
| **听感打磨**   | 独立“磨耳朵”黑胶唱片模式（顺序/随机/单曲循环）；背景搭配 12 种水墨风格并自动识别深色底生成反色保护。        |
| **分享与离线** | 支持 PWA 技术，可安装为桌面应用离线独立轻享；支持一键生成并长按保存高清“诗词水墨海报”。                     |

## 🛠️ 技术栈与架构设计

本项目最初是一个单文件巨石结构，后期经过彻底重构，目前采用如下现代化轻量级架构：

- **核心路线**：原生 Vanilla JS (ES Modules) + HTML5 + CSS3，**零运行时依赖**。
- **构建工具**：[Vite](https://vite.dev/)，提供极速 HMR 和优化的生产环境打包。
- **离线机制**：Service Worker (PWA) 支持网络优先/缓存回退策略。
- **组件结构**：由统一全局状态管理 (`app-state.js`) 驱动 UI (`poem-list-view`, `poem-detail-view`, `listen-view`) 更新，松耦合设计。
- **存储与语音**：数据全部留存于本地 LocalStorage；使用原生 Web Speech API 进行自然交互。
- **持续集成**：GitHub Actions 全自动验证脚本并部署至 GitHub Pages。

## 🚀 开发与部署

```bash
# 克隆项目
git clone https://github.com/meicai/inkverse.git
cd inkverse

# 安装仅用于构建校验的 devDependencies
npm install

# (可选) 运行古诗数据健康检查
npm run validate

# 启动冷更快的 Vite 本地原生开发服务器
npm run dev

# 编译代码为静态产物（输出到 dist/ 目录下）
npx vite build
```
